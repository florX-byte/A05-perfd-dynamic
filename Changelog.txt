## 🔄 Changelog v3.0 (from v2.0)

### 🚀 Performance Core Overhaul
* **Mali GPU Hard Lock:** Mengunci clock GPU maksimal ke `1GHz` dan minimal ke `560MHz`.
* **Governor Engine:** Set GPU governor ke `performance` dengan kebijakan *coarse demand scheduling*.
* **Bypass Throttling:** Menonaktifkan total *DVFS*, *PPM*, dan kebijakan *fallback frequency* bawaan vendor.
* **Input Booster Tuning:** Mengoptimalkan nilai `head`, `tail`, dan `level` pada respon *touch*, *multitouch*, dan *keyboard* untuk mereduksi *touch latency*.

### 🧠 CPU & Scheduler Optimization
* **Global Core Allocation:** Memaksa seluruh grup CPUSet untuk menggunakan semua core (`cpus=0-7`).
* **Low Latency Engine:** Menonaktifkan *load balancing* di setiap grup untuk mengurangi interupsi penjadwalan.
* **Util Clamp Injection:** Mengaktifkan `uclamp.latency_sensitive = 1` pada seluruh grup proses dan menyuntikkan nilai UClamp maksimal/minimal ke `1024` khusus untuk grup *top-app*.
* **Governor Tweaks:** Mengoptimalkan *Sugov_ext* dengan parameter batas atas dan bawah (*up/down rate limits*) yang agresif.

### 🧹 Boot Clean & Logging System
* **Logger Debloat:** Mematikan dan menghentikan secara permanen layanan `logd`, `logcat`, serta seluruh log *MediaTek* (`connsyslogger`, `emdlogger`, dll.) untuk menghemat siklus CPU.
* **Safe Injector Log:** Fungsi `write_val` kini mencatat status keberhasilan (✅/❌) penulisan node kernel langsung ke `/data/local/tmp/log_tweaks.txt`.

### 🎮 Game & App Priority Tuning
* **Task Profiles Injector:** Mengoptimalkan `task_profiles.json` untuk meningkatkan respon *cpuBoost*, *uclamp.max*, dan *task_runners* demi stabilitas FPS saat berpindah aplikasi.
* **Core Framework Boost:** Menaikkan prioritas `system_server` dan `SurfaceFlinger` secara agresif melalui penyesuaian nilai `renice`, `oom_score_adj`, dan *timerslack*.

### 🌐 Network & Storage I/O Enhancements
* **Network Latency:** Menggunakan kontrol kongesti `cubic` (atau `bbr` jika didukung kernel) dan mengaktifkan `tcp_low_latency = 1` untuk meminimalkan *ping jitter*.
* **DNS Override:** Mengunci DNS utama ke `1.1.1.1` dan DNS cadangan ke `8.8.8.8`.
* **Storage Pipeline:** Mengatur I/O Scheduler ke `bfq`, meningkatkan *read_ahead* ke `4096KB`, dan mengoptimalkan parameter antrean *iosched* untuk mempercepat *loading time*.

### 🌡️ Thermal & Throttle Unlock
* **Total Thermal Bypass:** Mematikan penguncian performa pada jalur `/thermal_zone*`, `/charger`, dan thermal HAL milik vendor (`vendor.thermal-hal`).
* **Charging Throttle Unlock:** Menghilangkan pembatasan performa dan pengisian daya saat ponsel terhubung ke pengisi daya (`sw_jeita`, `ap_thermal`, dll.).

### 📈 Final Boosting & FPS Lock
* **Display Smoothness:** Memaksa render commit rate pada `SurfaceFlinger` naik hingga `1.3x`.
* **Refresh Rate Lock:** Mengunci ambang batas minimal dan puncak layar secara konstan di `60Hz`.
* **Shader Cache Optimization:** Mengaktifkan *Skia shader cache* dan menonaktifkan *latch unsignaled* untuk visual UI yang lebih mulus tanpa *stutter*.

* Kompatibel penuh dengan **KernelSU**, **Magisk**, kernel berstatus *permissive*, serta pemicu boot manual via *Termux*.
