# A055-perfd-dynamic 🚀

Unified and dynamic performance engine for Samsung A05 (One UI 7, Android 15, Kernel 6.6) to optimize thread layout, I/O tweak, CPU and GPU optimization, and memory management.

## How to use?
1. Download the latest version from the Release tab or Telegram group.
2. Flash it via Magisk / KernelSU / APatch.
3. Reboot your device.

*Note: Check the GitHub Commit History or Release description for full changelogs.*
